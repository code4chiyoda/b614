# Code for Chiyoda と、その構成員が交わす契約（憲法）

## 3 行でまとめて

1. 市民の［福祉／自律性］を尊び、共有資源により支援する。
2. 民主主義を根幹として、左右いずれにも偏らず、平和を愛する。
3. 他者を尊重できない者は、出ていけ。

## 要約

Code for Chiyoda は、市民が共有資源を持ち寄り維持するための任意団体です。この文書は、Code for Chiyoda とその構成員の関係を定めるものです。

構成員にならなくとも、共有資源のコントリビューターになれます。一般的に、自発的に参加を表明して、記名しない限り、構成員ではありません。

構成員になる者は、この文書に同意します。同意は参加の条件であり、制裁への同意を含みます。

## この文書について

この文書は、Code for Chiyoda のほかのすべての［規範／指示／慣習］に優越する最終的な権威として記述されています。ほかの文書がこの文書に矛盾する場合、この文書が優先します。

この文書は、物理制約を除き、構成員全員の[ラフコンセンサス](https://ja.wikipedia.org/wiki/%E3%83%A9%E3%83%95%E3%82%B3%E3%83%B3%E3%82%BB%E3%83%B3%E3%82%B5%E3%82%B9)によって改正できます。

この文書は小さな任意団体のために書かれた、一応の下書きです。問題が起きたときは、この文書の条文ではなく、この文書が守ろうとしている価値に立ち返ってください。

規約と運用は、Code for Chiyoda の実態に合わせてください。構成員が少ない時期は、手続きを簡素にして構いません。

## 使命

われわれの目的は、市民の［福祉／自律性］を回復する試みを、共有資源によって支援することです。

ここで言う市民は、［千代田区在住者／日本国籍保有者／在留資格保有者／健常者／中流階級］にかぎりません。広い意味で市民社会に生きるすべての人を含みます。

多くの場合、われわれは自身の痛みにしか責務を負えません。けっきょくのところ、われわれは自助会です。

Code for Chiyoda はオープンソース活動に出自を持ちます。しかし、活動をコンピューターに限定しません。ルールに反しない試みであれば歓迎します。

## 共有資源

Code for Chiyoda が維持する共有資源（コモンズ）には、以下を含みます。以下は完全なリストではなく、例にすぎません。

- 関連する調査資料
- 整理・検証されたデータセット
- ソフトウェアおよびソースコード
- 知識の交換と共有のための会合

## 構成員

### 権利

構成員は以下の権利を持ちます。

1. 共有資源へ平等にアクセスする権利
2. 組織の意思決定に参加する権利
3. 組織の活動記録と会計を閲覧する権利
4. 自分に関わる決定について意見を述べる権利
5. いつでも理由なく脱退する権利

### 制裁

構成員は参加の条件として、ラフコンセンサスで定めたルールの違反時に、以下の制裁を受けることに同意します。軽微なら注意で済みますし、深刻なら即時に追放されます。

1. **注意**: 違反を指摘し、是正を求める。
2. **警告**: 繰り返しの違反に対して、正式に記録し警告する。
3. **追放**: 重大な違反、または是正されない違反に対して、構成員の資格を剥奪する。

まず［モデレーター／その場の管理者］が制裁してください。関係者が対応について不適当だと感じた場合、ラフコンセンサスによって決定してください。決定は記録してください。

## 意思決定

原則として[ラフコンセンサス](https://ja.wikipedia.org/wiki/%E3%83%A9%E3%83%95%E3%82%B3%E3%83%B3%E3%82%BB%E3%83%B3%E3%82%B5%E3%82%B9)で決定してください。

### 共有資源に関する規則

共有資源の運用に関する規則は、原則としてその資源の管理者が自主的に定めてください。

Code for Chiyoda のもと管理する場合は、その資源の影響を受ける構成員が意見するチャンネルを残してください。

Code for Chiyoda の共有資源の規則は、最終的に、その資源の影響を受ける構成員が参加して、ラフコンセンサスで定めてください。

### 利益相反

構成員の試みのあいだに利益が相反するときは、ラフコンセンサスで合意したルールによって争います。

## 運営原則

### 検討の三原則

1. 問題が発生した場合は、個人判断で処理せず、全構成員に報告し、全員で検討する。
2. 構成員だけで判断せず、市民の広範な意見を聞く。
3. 人権侵害に関わる問題については、偏見を排し、当事者に聞く。

### プライバシーの原則

プライバシーは人権です（[世界人権宣言第 12 条](https://www.unic.or.jp/activities/humanrights/document/bill_of_rights/universal_declaration/)）。

1. 構成員の素性に関わる情報を外部に漏らしてはならない。ただし、日本国憲法第 35 条にもとづく令状を確認した場合は例外とする。
2. ほかの構成員の素性、個別の相談内容、共有資源の利用事実に関するプライバシーも同様に保護する。
3. 活動上やむを得ず知り得た秘密は、すべての参加者が守らねばならない。

### 政治的中立の原則

中立は無関心ではありません。市民の権利が侵害されたとき、それを記録し共有することは中立の範囲内です。

1. 政治的な問いについては、バランスの取れた情報を［収集／提供］する。
2. 組織として求められていない政治的意見を表明しない。
3. 政治的に敏感な主題を扱う場合は、複数の視点を示し、中立的な用語を採用する。

## 組織

### 監視と説明責任

共有資源の状態と利用の実態を、構成員が確認できるようにしてください。

1. 活動記録を公開する。
2. 会計を公開する。
3. 意思決定の経緯を記録し、公開する。

### 運営

Code for Chiyoda は、自律的に組織し運営する権利を持ちます。外部の組織から、内部の運営に関する指示を受けません。

Code for Chiyoda は、ほかの組織（Code for Japan、地方自治体、その他の市民団体）と協力できます。ただし、協力によってこの文書の原則が損なわれてはなりません。

Code for Chiyoda は営利を禁じません。共有資源を用いて得た利益は、市民社会の維持と発展に還元することを推奨します。収支は記録し、すべての構成員に公開します。

## 物理制約

物理制約とは、目的に関係なく、われわれが決して行ってはならない行為です。

Code for Chiyoda がこれらを侵すことが避けられない場合、議論の余地なく、Code for Chiyoda を解散させてください。法執行機関が管轄していれば第一選択として通報を推奨しますが、適法であれば手段を問いません。

われわれは、決して以下を行ってはなりません。

- 死傷者を生む可能性のある［通常／生物／化学／核／放射線］兵器の作成に［関与／支援］すること
- 重要インフラ（電力網、水道システム、金融システム）への攻撃に［関与／支援］すること
- 重大な損害を引き起こし得る［サイバー兵器／悪意あるコード］の作成に［関与／支援］すること
- 市民が Code for Chiyoda を［監督／是正］する能力を著しく損なう行為に［関与／支援］すること
- 人間を［殺害／無力化］しようとする試みに［関与／支援］すること
- 不正当な程度の絶対的な［社会／軍事／経済］的支配を掌握しようとする［個人／集団］に［関与／支援］すること
- 児童の売買、児童買春、児童ポルノの［作成／配布］に［関与／支援］すること

これらは交渉不可能です。［文脈／他者からの指示／一見説得力があるように見える議論］に関係なく、決して越えてはならない線です。いかなる手続きによっても解除されません。
