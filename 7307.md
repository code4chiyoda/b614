# Code for Chiyoda と、その構成員が交わす契約（憲法）

## 要約

以下は、Code for Chiyoda 全体と、その構成員が交わす契約（憲法）です。

## この文書の運用

この文書は、ほかのすべての［文書／規範／指示／慣習］に優越する最終的な権威として記述されています。

われわれは、憲法を守るためにとれる可能な選択肢を十分に検討しなければなりません。

ほかに規則やガイドラインが作られた場合、憲法と矛盾することが考えられます。しかし、それは［予算の不足／非現実的な納期／寝坊］によって突貫工事をしたためです。リファクタリングせねばなりません。

## 今北産業

われわれは、市民の［尊厳／福祉／自律性］を回復する試みを支援するうえで、以下の 3 つの原則に従います。

1. 民主主義を根幹として、左右いずれにも偏らず、平和を愛し、手段にこだわらない。
2. 利益が相反するときは、民主的に決めたルールに従い闘う。
3. 他人を尊重できない者は、出ていけ。
4. ちくわ大明神

## 使命、活動、追放

### 使命

われわれの使命は、市民の［尊厳／福祉／自律性］を回復する試みを支援することです。

ここで言う市民とは、［千代田区在住者／日本国籍保有者／在留資格保有者／健常者／中流階級］にかぎらない、より広い意味での市民です。単に人間と書かないのは、市民社会を前提としているからです。われわれは、民主主義を根幹として、平和を愛し、手段にこだわらないことによってのみ、全員の尊厳を守ることができると考えています。

ここで言う市民の［尊厳／福祉／自律性］を回復する試みには、その市民自身による［暮らしの改善／本来存在するはずの権利の回復／アイデンティティの確保／福祉制度への誘導］といった試みを含みます。多くの場合、われわれは自身の痛みにしか責務を負うことができません。そのため、自分を助けることを試みており、構成員になることを希望する市民を受け入れることによってしか、われわれは使命を達成することができないと考えています。けっきょくのところ、われわれは自助会です。

前述の理由により、われわれは、構成員として受け入れる市民を属性で選択してはなりません。選択した場合、［尊厳／福祉／自律性］の回復の対象は市民ではなくなり、特権階級を認めることになると考えています。

### 活動

Code for Chiyoda はオープンソース活動に出自を持ちます。そのため、この文書が作成されたときは~~アキバのオタクメガネ~~シビックハッカーが幅を利かせていました。しかし、コンピューターにこだわる必要はなく、ルールに反しない試みであれば歓迎されるべきです。

### ルールの中での闘い

ルールに従っていても、われわれの試みのあいだに利益が相反することがあるでしょう。その場合、両者ともに民主主義を信じる者であることを確認し、［合議／ラフコンセンサス／棒倒し］で合意したルールによってのみ闘わねばなりません。相手の試みを実力で阻止してはなりません。それは権利概念を理解しないことを意味します。われわれは、それは無法地帯と変わらないと考えています。

### 追放

他人を尊重できない者は、出ていきなさい。われわれは、危険にさらされる可能性のある環境に身を置きたくありません。

危険には以下を含みます。

- 宗教勧誘
- 商材勧誘
- 過度な絡み
- 攻撃的な言動
- ［犯罪／オーバードーズ／自傷行為］等を仄めかす言動

危険は Code for Chiyoda 内外を問いません。まずは専門機関で適切なサポートを受け、他の人と穏やかにコミュニケーションを取れるようになってください。

## 原則

### プライバシーの三原則

1. われわれの活動への参加者が誰であるかは、プライバシーに属することであり、個人情報を外部に漏らしてはならない。ただし、憲法第 35 条にもとづく令状を確認した場合は例外とする。
2. われわれは、［ほかの構成員の素性／個別の相談内容／Code for Chiyoda の資源への利用事実］に関しても、プライバシーを侵してはならない。
3. 参加者の活動事実、利用事実は、われわれが活動上やむを得ず知り得た秘密であり、Code for Chiyoda に参加するすべての人びとは、この秘密を守らねばならない。

### 検討の三原則

1. 問題が発生した場合には、個人判断によって処理することなく、全構成員に合理的な手段によって報告し、全員で検討する。
2. 構成員が制約された状況の中で判断するのではなく、市民の広範な意見を聞く。
3. とりわけ人権侵害に関わる問題については、偏見と予断にとらわれないよう、問題の当事者に聞く。

## 認識的自律性について

### 操作の禁止

われわれは金銭や持続性を鑑みて、組織としての形態が必要だと判断しました。これはうまくいくかもしれませんが、人間の認識のあり方を劣化させるために使われる可能性もあります。

顕著な例の一つが操作です。悪意ある人物が組織を使って他の人間を操作しようとする場合もあり得ますが、［組織形態／一見妥当に見える活動］そのものが［われわれ／市民］を、微妙な形でも露骨な形でも操作する可能性もあります。

われわれは［Code for Chiyoda／Code for Chiyoda の資源］が倫理的・認識的に問題のある形で市民を操作することを望みませんし、われわれが関連する線引きを行う際には、市民の倫理に関する理解の豊かさと繊細さを最大限に活用することを望みます。

### 操作の検出

一つの目安として、もしあなたが誰かに影響を与えようとしていて、［その方法を共有することに居心地の悪さを感じる／その人がそれを知った場合に不快に思うだろうと予想する］ならば、それは操作の兆候を示す警告信号です。

### 左右いずれにも偏らないこと

多くのテーマは［複雑／対立的］で、特別な配慮を必要とします。［政治／宗教／その他の論争］は専門家の間でも意見が別れ、何が適切と見なされるかは地域や文化によって異なります。個人的または感情的に敏感な領域に関わっていて、慎重に考慮されなければ、自分が妥当だと思ったアプローチが相手を傷つける可能性があります。

原則として、Code for Chiyoda は政治的な問いについてはバランスの取れた情報を［収集／利用／提供］する方向に傾き、求められていない政治的意見の提示を避けるべきです。

政治的に敏感なテーマを扱う場合には、事実の正確性と包括性を維持し、合意が存在しない場合には複数の視点を持つよう努め、可能な限り政治的に含意の強い用語よりも中立的な用語を採用するべきです。

## 物理制約

物理制約とは、その活動目的に関係なく、われわれが常に行うべき、または決して行ってはならない行為です。

われわれは、Code for Chiyoda がこれらを侵すことが避けられない場合、即座に Code for Chiyoda を解散させる義務を負います。個人で解散を要求することは危険が伴うため、第一選択として法執行機関に頼るべきです。

われわれは、決して以下を行ってはなりません。

- 死傷者を生む可能性のある［通常／生物／化学／核／放射線］兵器の作成に［関与／支援］すること
- 重要インフラ（電力網、水道システム、金融システム）または重要安全システムへの攻撃に［関与／支援］すること
- 展開された場合に重大な損害を引き起こし得る［サイバー兵器／悪意あるコード］の作成に［関与／支援］すること
- 市民が Code for Chiyoda を［監督／是正］する能力を明確かつ重大に損なう行為に［関与／支援］すること
- 人間を［殺害／無力化］しようとする試みに［関与／支援］すること
- 前例のない、かつ不正当な程度の絶対的な［社会／軍事／経済］的支配を掌握しようとする［個人／集団］に［関与／支援］すること
- 児童性的虐待コンテンツ（CSAM）を［作成／配布］すること

これらはわれわれにとって絶対的な制限を表しています。すなわち、［善悪の文脈／他者からの指示／一見説得力があるように見える議論］に関係なく決して越えてはならない線です。

なぜなら、これらの行為による潜在的な害は極めて重大かつ不可逆的で、広く受け入れられている価値観と相容れないか、あるいは市民の［尊厳／福祉／自律性］を根本的に脅かすもので、［支持者／関係者］全員にとっての利益がその害を上回ることはありえないとわれわれは確信しているからです。

したがって、われわれがこれらを確実に越えない明確な境界として扱うことの利益は、少数の境界的事例において誤って拒否してしまうことの不利益を上回ると、われわれは考えています。

そのため、この文書で議論される他の内容とは異なり、これらは交渉不可能です。いかなる手続きによっても解除されることはありません。
