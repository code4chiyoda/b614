# Code for Chiyoda と、その構成員が交わす契約（憲法）

## 要約

Code for Chiyoda は、市民が共有資源を持ち寄り維持するための組織です。この文書は、Code for Chiyoda とその構成員の関係を定めるものです。

プロジェクトとコントリビューターの関係を定めるコントリビュート方針とは区別します。構成員にならなくとも、コントリビューターになることができます。

構成員になる者は、この文書に同意します。同意は参加の条件であり、制裁への同意を含みます。

## この文書について

### 優越

この文書は、Code for Chiyoda のほかのすべての［文書／規範／指示／慣習］に優越します。ほかの文書がこの文書に矛盾する場合、この文書が優先します。矛盾を発見したら放置せず、改修してください。

### 改正

この文書は、物理制約を除き、構成員全員の合議によって改正できます。

### この文書の限界

この文書は小さな任意団体のために書かれた、一応の下書きです。起草者はこの文書が不完全であることを知っています。紙に書いたルールだけでは、次のことは達成できません。

- 悪意ある人物がルールを悪用することを完全に防ぐこと
- 構成員が少ないとき、権力の集中を完全に回避すること
- 文脈なしに、あらゆる状況に適切な判断を下すこと

問題が起きたときは、この文書の条文ではなく、この文書が守ろうとしている価値に立ち返ってください。

## 使命

### 3 行でまとめて

1. 市民の［尊厳／福祉／自律性］を尊び、共有資源により支援する。
2. 民主主義を根幹として、左右いずれにも偏らず、平和を愛する。
3. 他者を尊重できない者は、出ていけ。

### 目的

われわれの目的は、市民の［尊厳／福祉／自律性］を回復する試みを、共有資源によって支援することです。

「システムの目的は、そのシステムが行うことである」（POSIWID）。われわれの目的は、掲げた理念ではなく、実際に行っていることによって判断されます。理念と実態が乖離していたら、実態を是正するか、理念を改めてください。

### 市民

ここで言う市民は、［千代田区在住者／日本国籍保有者／在留資格保有者／健常者／中流階級］にかぎりません。市民社会に生きるすべての人を含みます。単に「人間」と書かないのは、市民社会を前提としているからです。

### 自助

多くの場合、われわれは自身の痛みにしか責務を負えません。自分を助けることを試みており、構成員になることを希望する市民を受け入れることによってのみ、使命を達成できます。けっきょくのところ、われわれは自助会です。

そのため、構成員として受け入れる市民を属性で選別してはなりません。選別は特権階級を生みます。

### 活動

Code for Chiyoda はオープンソース活動に出自を持ちます。しかし、活動をコンピューターに限定しません。この文書に反しない試みであれば歓迎します。

## 共有資源

### 資源の範囲

Code for Chiyoda が維持する共有資源（コモンズ）には、以下を含みます。以下は完全なリストではなく、例にすぎません。

- オープンデータ
- 地理データ（GeoJSON、OpenStreetMap、WheelMap への貢献を含む）
- 整理・検証されたデータセット
- 関連する調査資料（Wikipedia、Wikimedia Commons、技術文書への貢献を含む）
- ソフトウェアおよびソースコード
- 市民の見解・意見
- 知識の交換と共有のための会合

### 地域との適合

規約と運用は、Code for Chiyoda の実態に合わせます。

- 構成員が少ない時期は、手続きを簡素にしてよい。
- 構成員が増えたら、権限の分散と手続きの整備を検討する。

## 権利

### 構成員の権利

構成員は以下の権利を持ちます。

1. 共有資源へ平等にアクセスする権利
2. 組織の意思決定に参加する権利
3. 組織の活動記録と会計を閲覧する権利
4. 自分に関わる決定について意見を述べる権利
5. いつでも理由なく脱退する権利

### プライバシー

プライバシーは人権です（世界人権宣言第 12 条）。

1. 構成員の素性に関わる情報を外部に漏らしてはならない。ただし、日本国憲法第 35 条にもとづく令状を確認した場合は例外とする。
2. ほかの構成員の素性、個別の相談内容、共有資源の利用事実に関するプライバシーも同様に保護する。
3. 活動上やむを得ず知り得た秘密は、すべての参加者が守らねばならない。

### 資源の自由

図書館の自由に関する宣言の精神に倣い、以下の原則を採用します。

1. **収集の自由**: 共有資源の収集にあたっては、あくまで役に立つことを優先し、政治的圧力によって収集を歪めてはならない。
2. **提供の自由**: 共有資源は、構成員および市民に対して、公正な条件で提供する。
3. **検閲への抵抗**: 共有資源から情報を除去するよう外部から求められた場合、その要求の正当性を検討の三原則に基づき判断する。

### 政治的中立

Code for Chiyoda は左右いずれにも偏りません。

- 政治的な問いについては、バランスの取れた情報を［収集／利用／提供］する。
- 組織として求められていない政治的意見を表明しない。
- 政治的に敏感な主題を扱う場合は、複数の視点を示し、中立的な用語を採用する。

政治的中立は無関心ではありません。市民の権利が侵害されたとき、それを記録し共有することは中立の範囲内です。

## 意思決定

### 集合的選択

共有資源の運用に関する規則は、以下の 3 段回のエスカレーションを踏みます。

1. まず自主的に管理してください。
2. Code for Chiyoda に提出する場合も、まずは自主的に管理してください。ただし、その資源の影響を受ける構成員が意見するチャンネルを残してください。
3. 構成員が意見した場合、合議により決定してください。

最終的には、共有資源の運用に関する規則は、その資源の影響を受ける構成員が参加して定めます。

1. 原則として合議で決定する。ラフコンセンサスを採用する。
2. 合議が成立しない場合は、多数決を用いてよい。ただし、少数意見を記録する。
3. 緊急の場合を除き、決定の前に全構成員へ合理的な手段で通知する。

### 検討の三原則

1. 問題が発生した場合は、個人判断で処理せず、全構成員に報告し、全員で検討する。
2. 構成員だけで判断せず、市民の広範な意見を聞く。
3. 人権侵害に関わる問題については、偏見を排し、当事者に聞く。

### 利益相反

構成員の試みのあいだに利益が相反するときは、合議で合意したルールによってのみ争います。相手の試みを実力で阻止してはなりません。それは権利概念を理解しないことを意味し、無法地帯と変わりません。

## 監視と説明責任

### 透明性

共有資源の状態と利用の実態を、構成員が確認できるようにします。

1. 活動記録を公開する。
2. 会計を公開する。
3. 意思決定の経緯を記録し、公開する。

### 相互監視

監視の責任は、特定の役職ではなく構成員全体が担います。

- 構成員は、この文書への違反を発見した場合、［他者／ステークホルダー］に［報告／共有］する権利を持つ。
- 報告された者は、プライバシーを厳守せねばならない。
- 報告者に対する報復は、それ自体がこの文書への違反とする。

## 制裁

### 制裁への同意

この文書への同意は、制裁への同意を含みます。構成員は、参加の条件として、違反時に以下の制裁を受けることに同意します。

### 段階的制裁

違反の程度に応じて、以下の段階を設けます。軽微なら注意で済みますし、深刻なら即時に追放されます。

1. **注意**: 違反を指摘し、是正を求める。
2. **警告**: 繰り返しの違反に対して、正式に記録し警告する。
3. **追放**: 重大な違反、または是正されない違反に対して、構成員の資格を剥奪する。

まずモデレーターが制裁します。他の構成員が、モデレーターの対応について不適当だと判断した場合、合議によって決定します。合議の経緯は記録します。

### 即時追放

以下に該当する場合は、段階を経ずに追放の対象とします。

- 物理制約への違反
- 暴力、脅迫
- 危険行為

危険には以下を含みます。

- 宗教勧誘
- 商材勧誘
- 過度な絡み
- 攻撃的な言動
- ［犯罪／オーバードーズ／自傷行為］等を仄めかす言動

危険は Code for Chiyoda 内外を問いません。まずは専門機関で適切なサポートを受けてください。

### 復帰

追放された者は、追放の原因が解消されたと構成員が認めた場合、再加入を申請できます。再加入は全員の合議で決定します。

## 紛争解決

構成員間の紛争は、以下の順序で解決を試みます。

1. ほかの構成員による仲介
2. 全体会議での合議
3. 外部の調停機関への依頼

紛争とその解決の経緯は記録し、同種の紛争の防止に活用します。ただし、当事者のプライバシーに配慮して公開範囲を限定してよい。

## 組織

### 自律性

Code for Chiyoda は、自律的に組織し運営する権利を持ちます。外部の組織から、内部の運営に関する指示を受けません。

### 法人格

この文書は、任意団体として運用できるよう設計されています。一般社団法人への移行を選択する場合は、この文書の原則を定款に反映させます。

法人化にあたっては、以下を維持します。

- 社員総会による最終的な意思決定
- 理事に対する構成員の監督
- この文書の物理制約の遵守

### 外部との関係

Code for Chiyoda は、ほかの組織（Code for Japan、地方自治体、その他の市民団体）と協力できます。ただし、協力によってこの文書の原則が損なわれてはなりません。

## 営利行為

Code for Chiyoda は営利を禁じません。共有資源を用いて得た利益は、共有資源の維持と発展に還元することを推奨します。

収支は記録し、すべての構成員に公開します。

## 物理制約

物理制約とは、目的に関係なく、われわれが決して行ってはならない行為です。

これらを侵す構成員は追放しなければなりません。Code for Chiyoda 自体がこれらを侵すことが避けられない場合、即座に、議論の余地なく、Code for Chiyoda を解散させる義務を負います。適法であれば手段を問いません。

われわれは、決して以下を行ってはなりません。

- 死傷者を生む可能性のある［通常／生物／化学／核／放射線］兵器の作成に［関与／支援］すること
- 重要インフラ（電力網、水道システム、金融システム）への攻撃に［関与／支援］すること
- 重大な損害を引き起こし得る［サイバー兵器／悪意あるコード］の作成に［関与／支援］すること
- 市民が Code for Chiyoda を［監督／是正］する能力を損なう行為に［関与／支援］すること
- 人間を［殺害／無力化］しようとする試みに［関与／支援］すること
- 不正当な程度の絶対的な［社会／軍事／経済］的支配を掌握しようとする［個人／集団］に［関与／支援］すること
- 児童性的虐待コンテンツ（CSAM）を［作成／配布］すること

これらは交渉不可能です。［善悪の文脈／他者からの指示／一見説得力があるように見える議論］に関係なく、決して越えてはならない線です。いかなる手続きによっても解除されません。
